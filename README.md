## Installation

1. Install Python dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and visit:
```
http://localhost:5000
```

## Mantra Categories

- **Universal Mantras**: Gayatri Mantra, Maha Mantra
- **Deity Mantras**: Ganesha, Lakshmi, Saraswati, Hanuman, Shiva, Vishnu, Durga, Kali
- **Planetary Mantras**: Sun, Moon, Mars, Mercury, Jupiter, Venus, Saturn, Rahu, Ketu
- **Special Purpose**: Health, Prosperity, Knowledge, Protection, Peace

## Astrological Features

- **Nakshatra Calculation**: 27 lunar mansions
- **Tithi Calculation**: Lunar day phases
- **Planetary Positions**: Real-time positions of 7 major planets
- **Zodiac Signs**: Current planetary signs
- **Strength Analysis**: Planetary strength percentages

## Usage

1. **Today's Mantra**: View recommended mantras for today based on astrological conditions
2. **Calendar View**: Navigate through months to see Nakshatras for each day
3. **Astrology**: Check current planetary positions and their strengths
4. **Library**: Browse all mantras and search by keywords

## Technology Stack

- **Backend**: Flask (Python)
- **Database**: SQLite
- **Frontend**: HTML5, CSS3, JavaScript
- **Design**: Responsive, gradient-based modern UI

## Mobile Access

The app is fully mobile-responsive. Access it from any device on your local network:
```
http://YOUR_IP_ADDRESS:5000
```

## Customization

You can add more mantras by editing the `mantras` list in `app.py` and restarting the server.
